Exercise tracker

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

Get exercises

GET /api/exercise/log

POST to create a new user: POST /api/exercise/new-user?{username}

POST to add an exercise: POST /api/exercise/add?{userId}[&description][&duration][&date]

GET users's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

GET all users in database: GET /api/exercise/users

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Example: https://localhost:3000/api/exercise/add?userId=mwiyC&description=running&duration=45

Example: https://localhost:3000/api/exercise/log?userId=mwiyC&from=2012-01-10&to=2019-05-01&limit=2
